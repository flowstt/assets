# Assets

Shared resources for the FlowSTT project including images, icons, media files, and other static assets.

## Usage

Assets in this repository are referenced by the other FlowSTT subprojects (desktop app, website, specs) as needed. Keep files organized by type in subdirectories.

## Guidelines

- Use descriptive, lowercase, hyphenated filenames (e.g. `app-icon-512.png`).
- Prefer SVG for icons and logos where possible.
- Optimize raster images before committing (compress PNGs, use appropriate JPEG quality).
- Do not store generated or build-output files here.
